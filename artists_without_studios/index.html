Mess with Pixelated Reddit Images, Make A Painting in the Canvas Element

About

This website is an art project by me, Amanda Innis, who enjoys making "paintings" in the canvas element. I have also included imagery from Reddit's top stories for inspiration as well as filters to play with.

What this Site Can Do

Reddit files can be played with using the filters and changing the background color (click on the input itself to change color). Canvas imagery can be generated by clicking on the color input next to the brush size. You can change colors and brushes by clicking on each of the color inputs. And, if you want to take colors from the Reddit image, simply click on the input and, before using the color-picker, scroll up to the Reddit image, then click on color-picker and get your color.

As for capturing images, you can download a PNG file of the canvas drawing you make. Take it before you refresh the page--these castles are made of sand. However, taking a PNG of the Reddit image isn't possible so if you like what you make, just screenshot that one yourself.
