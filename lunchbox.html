Lunchbox (in progress)

This is an independent project with Marco Pariente-Cohen for the USDA This project is intended to replace the USDA’s paper application for free lunches for public school students. The paper application has confused users and prevented many qualified applicants from getting access to this program. We redesigned the sequence and presentation of information and created an online application to minimize confusion and enable more users to complete it successfully. My role: Research, design, illustration, wireframes and coding in HTML and CSS. Marco’s role: JS, Ruby and back-end development.
